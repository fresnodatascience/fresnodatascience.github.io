Prep week

Start here - Installing Python (Anaconda) and Your Code Editor

Data science requires the use of a programming langauge for parsing and manipulating information. The most popular, and the easiest, language is Python. For this course, we will be using Python 3.X.X( currently 3.7.3).

Python uses a host of different add ons for data science, which can be installed seperately or as a unified package. We are going to install it as a unified package, which will include the newest version of Python and all of the different data science libraries that Python can utilize. The name of this package is called Anaconda, and can be downloaded here.

If you need help with the installation on an Apple machine, this video should help.

If you need help with the installation on a Windows machine, this video should help.

If you're using a Linux machine, you probably don't need help.

In general, the Anaconda docs are a very useful resource for setup and startup. Check it out here.

If you are still having problems, head over to the "Contact" section of the site, and go to the Slack channel and ask for help there (it's the first circle button on the "Contact" page).

Once you have Anaconda and Python installed, head over to the next section!

Learning Python

Now that Python is available on your computer, you are offically a data scientist!

Just kidding...

Now that Python is available on your computer, you have to actually learn how to use it. Luckly, it is a very simple language syntacticly to learn.

For the absolute beginner, Mosh's Python tutorial is one of the best.

Another great resource is this.

For the more advanced user with little to no experience with Python, you can just watch this. This video skims over the language, so it should be faster to learn, and the Python version in the video is older, but everything else is the same.


This should be enough to start, but there is one more piece of advice that is universally used everyday and will help you in all aspects of this course and programming in general...

GOOGLE

Googling is the programmers most important tool because it is very likely that someone has had the same concerns as you and has asked someone else online. The result of this is a well written, well documented solution which is readily available online.

Googling is an art of sorts and will usually lead you to stackoverflow.com, which is an abundance of knowledge in web form.

As the old adage goes "Try, google, ask". This leads me to the final part of the prep week.

Ask for help

Since this is a community driven program, the help that one can recieve is largely related to the willingness of those in the know to help out. For this bootcamp, there is a slack channel, which would, ideally, have people from the group that are willing to help outside of bootcamp hours.

No one is forced to help another person, but it is highly encouraged because
a) You will learn better by teaching someone else.
b) You're being a good human being.

But ... no pressure.

The slack channel is located in the "Contact" section, or you can click here.

Good Luck!
